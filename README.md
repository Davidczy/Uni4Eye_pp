# Uni4Eye_++
This is the implementation of Uni4Eye++.

## Pre-trained model
The pre-trained model weights can be downloaded [here](https://drive.google.com/file/d/12Cw-OpyBcFbs2zcT368cG6DJPVKVRMEa/view?usp=sharing).

## Requirments
+ python >= 3.6
+ pytorch
+ timm
+ volumentations
+ nibabel
+ scikit-learn

## Usage
```
python main_pretrain.py
```
